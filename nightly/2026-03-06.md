## Phase 5: Smart Alerts Page

The dashboard now has an Alerts page that shows attorneys what needs their attention right now. Instead of checking each case one by one, they can see everything that's overdue, stale, or waiting for work in one place.

**What was built:**
- New `/alerts` page with three alert categories: overdue deadlines (red), stale cases with no updates in 30+ days (yellow), and paid contracts where case work hasn't started (amber)
- Each category shows a count and up to 50 items with client names, board types, and how many days overdue or stale
- Attorney filter chips let you narrow down to a specific attorney's cases
- Severity filter chips let you focus on just critical, warning, or info alerts
- The landing page now has a 6th KPI card showing total alert count, clickable to go straight to the alerts page
- Sidebar navigation updated — Alerts link is now active

**Bug fixes (pre-Phase 5):**
- Fixed a runtime crash on the Clients tab when the API returned no data
- Fixed snapshot card popovers rendering behind the tabs bar (z-index stacking issue)

**Why it matters:** Attorneys can now spot problems before they become emergencies. Overdue deadlines and stale cases are surfaced automatically instead of requiring manual case-by-case review.
